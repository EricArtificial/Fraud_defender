# Phishing Simulator (Static Version)

This is a pure HTML/CSS/JS version of the Phishing Simulator. It demonstrates a "Shell Website" (MITM) attack scenario for educational purposes.

## How to Run

Simply open `index.html` in your web browser.

No build steps or dependencies are required.

## Structure

- `index.html`: The main structure of the application.
- `style.css`: Custom styles (mostly Tailwind CSS via CDN).
- `script.js`: Application logic (state management, simulation, mock AI response).

## Features

- **Landing Page**: Simulates a fake "Free VIP" claim page to lure users.
- **Simulation**: Shows what happens behind the scenes when a user enters their phone and OTP.
- **Educational Content**: Explains the attack mechanism and provides prevention tips.
